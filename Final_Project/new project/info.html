5 DAYS boot camp on HTML and CSS

Timeline: 12/08 to 18/08

12/08 - Day 1: Introduction to Basic HTML

On the first day, the session started with an introduction to basic HTML elements such as <form>, <container>, <label>, <list>, and <anchor> tags. Real-time examples were provided to demonstrate how these elements work. The challenge given was to create a resume using these basic HTML components.

13/08 - Day 2: Advanced HTML Concepts

On the second day, the focus shifted to more advanced HTML elements like <fieldset>, <legend>, and the <table> element along with their attributes. The challenge was to create a form and a table incorporating these advanced HTML features.

14/08 - Day 3: Introduction to CSS

Day three introduced the basics of CSS, including coloring, arranging elements, and linking stylesheets to HTML. The challenge for the day was to decorate a table horizontally and diagonally and create a simple website with two buttons: one for switching to light mode and another for dark mode.

15/08 - Day 4: Understanding Box Model and Flexbox

On the fourth day, the training covered the box model and flexbox in-depth, from basics to advanced concepts. The day's challenge was to create a web page using the box model and flexbox for layout.

16/08 - Day 5: Introduction to Bootstrap

The final day introduced Bootstrap and its components for rapid web development. The challenge was to create a complete website for the club using Bootstrap, incorporating what was learned throughout the week.
